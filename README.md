# 掘金 Markdown 主题列表

## 如何贡献

1. 创建一个 Git 仓库（可公开访问），添加主题样式 `index.css` 文件，并添加 License（需要兼容 MIT）后提交
2. Fork 此仓库并克隆到本地，执行 `git submodule init`
3. 添加第 1 步中创建的仓库为 submodule：`git submodule add [仓库地址] themes/[主题名字]`
4. 提交 pull request
